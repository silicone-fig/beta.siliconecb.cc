## install

1. clone the repo:
   ```bash
   git clone https://github.com/silicone-fig/beta.siliconecb.cc.git
   cd beta.siliconecb.cc
   ```
2. install dependencies:
   ```bash
   npm install
   ```

   if that fails, try:
   ```bash
   npm install --legacy-peer-deps
   ```

3. run the dev server:
   ```bash
   npm run dev
   ```
